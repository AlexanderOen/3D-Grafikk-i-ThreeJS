# 3D-Grafikk-i-ThreeJS
Dette prosjektet gikk ut på å lage en scene i ThreeJS med bruk av diverse avanserte teknikker, som:

- Bevegelse langs en ellipse
- Sprites
- Instansiering av objekter
- Light- og shadowmaps
- bump- og normalmaps
- Lava-refleksjoner og tåke
